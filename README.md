### Directories:

 1)  `src` -- Makefile and source files for two versions:
     i)  `sphf_all`
     ii) `sphf_scf`

 2)  `test_all` -- Test directory for testing the `sphf_all` code
     Copy executable `sphf_all` to directory `test_all`, then execute `test_all/sh_run`.
